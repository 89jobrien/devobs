# kgx Knowledge Graph

Project metadata and relationships stored in a kgx graph at .kgx/. devobs_ingest.crux parses YAML frontmatter from PROJECT notes, creates nodes per project and edges from project_relationships. Graph queryable via kgx search, kgx bfs, kgx wiki. 17 edge types defined in project.schema.json.
