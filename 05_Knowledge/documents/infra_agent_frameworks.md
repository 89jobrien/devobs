# Agent Frameworks

Three sibling Rust agent frameworks: braid (multi-agent orchestration), looprs (agent loop primitives), langchainx (LangChain-style chains in Rust). praxis observes all three. Supporting projects: rslm (LLM client abstractions), bamlish (BAML structured output), mcpipe (MCP protocol proxy).
