# Agent Framework Triad

Three sibling Rust agent frameworks approaching multi-agent
orchestration from different angles.

## Members

- **[[braid]]** — Multi-agent orchestration platform
- **[[looprs]]** — Agent loop primitives
- **[[langchainx]]** — LangChain-style agent chains in Rust

## Supporting Projects

- **[[rslm]]** — LLM client abstractions
- **[[bamlish]]** — BAML structured output
- **[[mcpipe]]** — MCP protocol proxy

## Observer

- **[[praxis]]** — observes all three as reference implementations
